## 🎨 Dark/Light Mode Class Pattern

### Background
```typescript
className="bg-black dark:bg-black light:bg-white"
```

### Text
```typescript
className="text-white dark:text-white light:text-gray-900"
```

### Cards
```typescript
className="bg-white/5 dark:bg-white/5 light:bg-white"
```

### Borders
```typescript
className="border-white/10 dark:border-white/10 light:border-gray-200"
```

### Input Fields
```typescript
className="
  bg-white/10 dark:bg-white/10 light:bg-white
  border border-white/20 dark:border-white/20 light:border-gray-300
  text-white dark:text-white light:text-gray-900
  placeholder-gray-500 dark:placeholder-gray-500 light:placeholder-gray-400
"
```
