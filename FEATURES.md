## 🕸️ Spider-Man Web Efekti

### Mouse Tracking
- Canvas tabanlı gerçek zamanlı çizim
- Mouse hareketlerini takip eden ağ çizgileri
- Fade-out animasyonu
- Performans optimize edilmiş
- Mix-blend-mode ile etkileyici görünüm

### Nasıl Çalışır
1. Mouse hareket ederken çizgiler oluşturur
2. Çizgiler yavaşça kaybolur
3. Maksimum 50 çizgi limiti ile performans korunur
4. Mavi gradient renkler

## 📦 Kurulum ve Kullanım

### Gereksinimler
Yeni özellikler için ek paket gerekmez. Mevcut dependencies yeterlidir.

### Başlatma
```bash
cd frontend
npm run dev
```

### Dil Değiştirme
1. Sağ üst köşedeki globe ikonuna tıklayın
2. İstediğiniz dili seçin
3. Sayfa otomatik yenilenir

### Chatbot Kullanımı
1. Sağ alt köşedeki chat butonuna tıklayın
2. Mesajınızı yazın ve Enter'a basın veya gönder butonuna tıklayın
3. Bot otomatik olarak yanıt verecektir
