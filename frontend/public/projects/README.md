# Proje Görselleri

Bu klasörlere proje görsellerinizi ekleyin. `projectsData.ts` içindeki `images` dizisinde dosya adlarını kullanın.

## Web projeleri (`web/`)
- **Yatay (landscape)** görseller kullanın (örn. 16:9 veya 3:2).
- Örnek dosya adları: `kurumsal-site-1.jpg`, `e-ticaret-1.png`

## Mobil projeleri (`mobile/`)
- **Dikey (portrait)** görseller kullanın (örn. 9:16 veya 3:4).
- Örnek dosya adları: `qr-menu-app-1.jpg`, `mobil-uygulama-1.png`

Desteklenen formatlar: JPG, PNG, WebP.
